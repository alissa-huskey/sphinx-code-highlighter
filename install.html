InstallÂ¶

  1. Install the sphinx-code-highlighter module via pip

    pip install sphinx-code-highlighter

    Or poetry

    poetry add sphinx-code-highlighter

  2. Add code_highlighter to the extensions array of your config.py:

    # conf.py
extensions = [
    "code_highlighter",
    # ...
]

    Or your Jupyter Book _config.yml:

    # _config.yml
sphinx:
  extra_extensions:
    - code_highlighter
    # ...

  3. Add sphinx-code-highlighter to your requirements.txt for publishing (for example to Google Pages.)

    # requirements.txt
jupyter-book
sphinx-code-highlighter
