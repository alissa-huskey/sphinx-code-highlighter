Usage¶

The code-block-hl directive works just like the code-block directive but with the added feature of partially highlighted lines.

Syntax¶

While the builtin version only allows you to highlight full lines via the :emphasize-lines: option, with this extension you can surround the parts of lines you want to highlight with !!!.

You can use the directive in reStructuredText documents…

.. code-block-hl:: python
  :linenos:
  :caption: The text surrounded by "!!!" gets highlighted.

  dwarves = [
    "Bashful",
    "Dopey",
    "Happy",
    "Grumpy",
    "Sleepy",
    "Sneezy",
    "Doc",
  ]

  i = 0
  while i < len(!!!dwarves!!!):
    !!!name!!! = dwarves[i]
    print(f"{name}s Room")
    i += 1

Or MyST Markdown documents.

```{code-block-hl} python
:linenos:
:caption: "The text surrounded by `!!!` gets highlighted."
dwarves = [
  "Bashful",
  "Dopey",
  "Happy",
  "Grumpy",
  "Sleepy",
  "Sneezy",
  "Doc",
]

i = 0
while i < len(!!!dwarves!!!):
  !!!name!!! = dwarves[i]
  print(f"{name}s Room")
  i += 1
```
